Analyze A/B-Test Results
Udacity Data Analyst Nanodegree, February 2019

Description
This project is an example of an A/B test run by an e-commerce website, checking if to add a new page to the website, basd on a conversion rate.
The experiment had two groups of participants, control and treatment, and two different pages were examined. The new one and the old one.

Topics covered with this project include Probability, A/B Testing and Logistic Statistics.
